#  Speech-to-Text Transcription

A simple Python project that converts **MP3 audio files** into text using the **SpeechRecognition** library and **Pydub**. Designed to run in **Google Colab**, this project allows users to upload audio files, convert them to WAV format, and transcribe speech using Google’s Speech Recognition API.

---

##  Features
- Upload MP3 files directly in Google Colab.
- Convert MP3 → WAV using **Pydub**.
- Transcribe audio to text with **SpeechRecognition** and Google’s API.
- Basic error handling for unclear or noisy audio.

---

##  Tech Stack
- **Python 3**
- **SpeechRecognition**
- **Pydub**
- **Google Colab**
